# SistemaNotas

## Descripcion:
<p align ="justify">
Este proyecto consiste en un sistema para registrar las 4 notas de un estudiante y sacar la calificacion final calculando el promedio de todas. Esta actividad busca comprender como usar las excepciones para detectar y prevenir errores, usando bloques try-cratch-finally, demostrando la funcion de las excepciones en el programa. Muestra lo importante que son las excepciones para anticipar y detectar errores, ademas ayudar a comprender el error y que lo causa.
</p> <br>
